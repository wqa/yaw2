YAW.

A small, private, end-to-end-encrypted mesh — chat & file sharing, peer-to-peer. The server only introduces you; it never sees your messages.

🌐 Web client — recommended, nothing to install Open the link your host shared with you in any modern browser (Chrome, Safari, Firefox, Edge), sign in, and follow the in-app guide. Your key stays in your browser.
⌨︎ Command-line client Python (aiortc + libsodium). Source & setup at github.com/wqa/yaw2 — needs Python 3.9+. Same network, same contact cards as the web client.

First run: pick a nickname, copy your contact card, swap cards with friends and Accept each other, then connect — and back up your key.
